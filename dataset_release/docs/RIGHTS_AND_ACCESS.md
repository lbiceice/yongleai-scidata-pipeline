# Rights And Access

The underlying mural photographs were produced within an institutional documentation context and are not redistributed in this open Scientific Data repository package. The public deposit therefore covers the annotation tables, manifests, taxonomies, statistics, prompts, figures, tables, and supporting documentation bundled here.

The open package is released under CC BY 4.0. Source panoramas and patch PNG images remain subject to institutional rights authorization and are handled outside this DOI package. This separation keeps the citable annotation layer public while preserving a controlled route for protected image assets.

## Public layer vs. protected image layer

This Zenodo record releases the public annotation-and-metadata layer:

- source manifests;
- patch manifests and crosswalks;
- STEP3 per-engine and consensus tables;
- auxiliary STEP4 damage tables;
- taxonomies, prompts, configuration snapshots, and validation statistics.

It does **not** release:

- full-resolution panoramic photographs;
- the 7,455 patch PNG files cropped from those photographs;
- local execution traces or institution-internal rights paperwork.

## Controlled-access request route

Requests for the protected image layer should be directed to the corresponding author:

- **Contact:** `lbice@gzgs.edu.cn`
- **Institution:** Guangzhou College of Technology and Business

To support review, requests should include:

1. requester name, affiliation, and contact details;
2. project title and intended use of the images;
3. whether access is for peer review, academic research, or another non-commercial scholarly purpose;
4. agreement not to redistribute protected image files without separate authorization.

The target for an initial response is **15 business days**. Reviewer access can be arranged through the submission route when journal editors request protected-image inspection during peer review.

Requests are intended to be checked only for verifiable identity, institutional affiliation, and acceptance of the non-redistribution terms required by the image-rights holder. They are not intended to be screened on research topic, institution type, or nationality beyond what is necessary to confirm legitimate scholarly use and compliance with the underlying rights restrictions.

The repository package also provides two companion documents so that this access route is not dependent on ad hoc email wording:

- `docs/IMAGE_ACCESS_REQUEST_TEMPLATE.md`
- `docs/DATA_USE_AGREEMENT_TEMPLATE.md`

These templates are intended to standardize request content, non-redistribution terms, and reviewer-access handling across future repository versions.

## Stewardship note

The public DOI record preserves the annotation-and-metadata layer, the rights-and-access note, and the request templates as part of the long-term repository documentation. If the corresponding-author contact changes in a future version, the access-route contact details should be updated in the repository metadata while preserving the same restrictions and request criteria.

## Future release path

If institutional authorization for public image redistribution is granted later, the patch image layer should be deposited as a separate image-layer record and cross-linked to DOI `10.5281/zenodo.19679501`.
