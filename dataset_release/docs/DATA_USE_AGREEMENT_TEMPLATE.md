## Covered material

This agreement applies only to the protected image layer that is not included in the open DOI package, including any approved source panoramas, patch PNG files, or later controlled-access image derivatives.

## Permitted use

The requester may use the approved files only for the non-commercial scholarly purpose described in the corresponding access request.

## Restrictions

The requester agrees that they will not:

1. redistribute the protected files to third parties;
2. post the protected files to public repositories, websites, or file-sharing services;
3. use the files for commercial exploitation without separate authorization;
4. remove or obscure attribution, provenance, or rights statements that accompany the files.

## Stewardship and deletion

Unless a different retention period is explicitly approved, the requester should delete local copies once the approved use is complete, or renew the request if continued retention is needed.

## Reviewer access

When access is required for journal peer review, the same non-redistribution terms apply. Reviewer access may be coordinated through journal editors when anonymous inspection is necessary.
